Verwertung von offenen Daten

Als Beispieldatensatz dient die Ideensammlung der Cooperation OGD Österreich:

Auszug aus dem Datensatz
"ID Sub_ID1 Sub_ID2 ""Text"""
"01 0 0 ""Uservoting für neue gewünschte Datensätze"""
"01 1 0 ""Verzeichnis (noch nicht) veröffentlichter Daten"""
"02 0 0 ""OGD positiv verkaufen"""
"02 1 0 ""In der Schule anfangen"""
"02 2 0 ""nicht nur auf Opendata -0 Seiten auf die Nutzung hinweisen (Bsp Parlament -0 OD Logo bei normaler Websuchmaske)"""
"02 3 0 ""Bedeutung über Initiativen (z B USA/Deutschland) aufklären"""
Beschreibung: Jede ID für eine Idee steht, die Sub_ID1 für einen Kommentar zu dieser sowie die Sub_ID2 für einen Kommentar zu einem Kommentar.

Die notwendigen Schritte:

  • Da sich die Daten nicht ändern, wurde das CSV lokal auf diesem Server hinterlegt.
  • Um die weitere Verarbeitung zu vereinfachen, wurde das CSV mit einem Editor (Suche und Ersetze) in folgendes Format gebracht:
"ID";"Sub_ID1";"Sub_ID2";"Text"
"01";"0";"0";"Uservoting für neue gewünschte Datensätze"
"01";"1";"0";"Verzeichnis (noch nicht) veröffentlichter Daten"
"02";"0";"0";"OGD positiv verkaufen"
"02";"1";"0";"In der Schule anfangen"
"02";"2";"0";"nicht nur auf Opendata -0 Seiten auf die Nutzung hinweisen (Bsp Parlament -0 OD Logo bei normaler Websuchmaske)"
"02";"3";"0";"Bedeutung über Initiativen (z B USA/Deutschland) aufklären"
  • Dieses CSV kann nun über einen GET-Call aufgerufen und verarbeitet werden.
  • Es wurde die jquery-csv Lib verwendet, um aus dem CSV die entsprechenden JSON Objekte zu generieren
  • Über eine for-Schleife wurden nun aus diesen Objekten die unten zu sehenden Panels generiert
  • Das wars auch schon, so einfach kann Open Data sein!
